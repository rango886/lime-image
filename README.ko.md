> **Lime Image Viewer**는 가볍고 세련된 디자인의 이미지 뷰어입니다. 다양한 포맷 지원과 강력한 메타데이터 편집 기능을 갖추고 있으며, 5가지의 편리한 확대/축소 모드와 사용자 친화적인 단축키를 통해 효율적이고 부드러우며 방해 없는 몰입형 이미지 감상 경험을 제공합니다.

---

## ✨ 핵심 기능

### 🎨 몰입형 시각 경험
- **프레임리스 디자인**: 복잡한 기존 윈도우 테두리를 제거하고, 메뉴바는 자동으로 숨겨지며 마우스를 올릴 때만 우아하게 나타납니다. 화면의 모든 픽셀을 오직 이미지를 위해 사용하세요.
- **부드러운 이미지 전환**: 깊이 최적화된 이미지 로딩 로직으로 끊김 없이 부드러운 전환을 제공합니다.

### 🔍 편리한 확대/축소 모드
다양한 상황에 맞는 5가지 스마트 줌 로직을 제공합니다 (**숫자 키 `1` - `5`로 빠른 전환 지원**):
1.  **스마트 자동 줌**: 이미지의 가로세로 비율을 지능적으로 인식합니다. 가로 이미지는 너비에, 세로 이미지는 높이에 우선 맞춰 창 안에 이미지가 잘리지 않고 표시되도록 합니다.
2.  **포커스 잠금 (비교 모드)**: 연속 촬영 사진 선별이나 틀린 그림 찾기에 최적화되어 있습니다. 왼쪽 상단 등 세부를 확대하여 관찰하다가 다음 이미지로 넘기면, 소프트웨어가 동일한 확대 비율과 위치를 자동으로 유지하여 반복적인 조작이 필요 없습니다.
3.  **센터 잠금**: 이미지 전환 후에도 현재 확대 비율을 유지하며, 항상 화면 중앙을 기준으로 렌더링합니다.
4.  **너비 우선**: 창 너비에 맞춰 이미지를 강제로 확대합니다 (웹툰이나 웹페이지 스크린샷 보기에 적합).
5.  **높이 우선**: 창 높이에 맞춰 이미지를 강제로 확대합니다.

### 🛠 강력한 기능 세트
- **다중 형식 메타데이터 편집**: 이미지 메타데이터를 보는 것뿐만 아니라 편집도 지원합니다.
- **빠른 창 크기 조절**: 단축키를 통해 소프트웨어 창 크기를 빠르게 조절하여 다양한 해상도의 데스크탑 레이아웃에 유연하게 대응합니다.
- **모든 포맷 지원**: png, jpeg, jpg, bmp, gif, tif, tiff, webp, ico, jfif, heic, heif, avif, jxl, svg, psd, psb, dds, tga, xcf, hdr, exr, dng, cr2, cr3, nef, arw, orf, rw2, raf, sr2, srw, pef, x3f, erf, mef, mos, mrw, 3fr, kdc, jp2, j2k, jpf, jpx, jpm, mj2, pbm, pgm, ppm, pnm, pam, pcx 지원
- **유연한 정렬 방식**: **파일명**, **수정 시간**, **파일 크기**에 따른 오름차순 또는 내림차순 정렬을 지원하여 파일 관리가 용이합니다.

### 🌍 다국어 지원
중국어(간체/번체), 영어, 일본어, 한국어, 프랑스어

## ⌨️ 단축키 가이드

탐색 효율을 높이기 위해 Lime Image Viewer는 풍부한 키보드 단축키를 제공합니다:

| 기능 분류 | 키 | 설명 |
| :--- | :---: | :--- |
| **기본 탐색** | `←` / `→` / `휠` | 이전 / 다음 이미지 전환 (길게 누르면 빠른 넘기기) |
| | `Home` | 폴더의 첫 번째 이미지로 이동 |
| | `End` | 폴더의 마지막 이미지로 이동 |
| | `Esc` | 창 닫기 |
| **뷰 컨트롤** | `+` / `-` | 소프트웨어 창 빠르게 확대 / 축소 |
| | `F11` | 전체 화면 모드 켜기 / 끄기 |
| | `L` | 현재 이미지가 있는 폴더 열기 및 파일 위치 확인 |
| | `I` | 이미지 메타데이터 패널 표시 / 숨기기 |
| **확대 모드** | `1` - `5` | 5가지 확대 모드 빠른 전환 (자동/포커스/센터/너비/높이) |
| **정렬 제어** | `N` / `Shift + N` | **이름**순 오름차순 / 내림차순 |
| | `T` / `Shift + T` | **시간**순 오름차순 / 내림차순 |
| | `S` / `Shift + S` | **크기**순 오름차순 / 내림차순 |
