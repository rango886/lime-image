> **青柠图片** 是一款轻量级、高颜值图片查看器。它集多格式支持与强大的元数据编辑于一身，有五种方便的缩放模式，配合人性化的快捷键操作，为您带来高效、流畅且无干扰的沉浸式看图体验。

---

## ✨ 核心特性

### 🎨 沉浸式视觉体验
- **无边框设计**: 摒弃繁杂的传统窗口边框，菜单栏自动隐藏，鼠标悬浮时优雅浮现。让屏幕的每一像素都为图片服务。
- **流畅的图片切换**: 深度优化的图片加载逻辑，丝般顺滑，拒绝卡顿。

### 🔍 方便的缩放模式
青柠图片提供 5 种智能缩放逻辑，满足不同场景的浏览需求（**支持通过数字键 `1` - `5` 快速切换**）：
1.  **智能自动缩放**: 智能识别图片长宽比。横图优先适应宽度，竖图优先适应高度，确保图片始终完整展示在窗口内。
2.  **焦点锁定缩放 (对比模式)**: 专为连拍挑选或找茬设计。当您在左上角放大观察细节时，切换至下一张图片，软件将自动保持相同的缩放比例和视觉位置，无需重复调整。
3.  **中心锁定缩放**: 切换图片后保持当前的缩放比例，始终以画面中心为基准进行渲染。
4.  **宽度优先**: 强制缩放图片以填满窗口宽度（适合阅读长条漫或网页截图）。
5.  **高度优先**: 强制缩放图片以填满窗口高度。

### 🛠 强大的功能集
- **多格式元数据编辑**: 不仅支持查看，还支持编辑图片的元数据。
- **窗口大小快速调整**: 支持通过快捷键快速调整软件窗口尺寸，灵活适应不同分辨率的桌面布局。
- **全格式支持**: 支持 png、jpeg、jpg、bmp、gif、tif、tiff、webp、ico、jfif、heic、heif、avif、jxl、svg、psd、psb、dds、tga、xcf、hdr、exr、dng、cr2、cr3、nef、arw、orf、rw2、raf、sr2、srw、pef、x3f、erf、mef、mos、mrw、3fr、kdc、jp2、j2k、jpf、jpx、jpm、mj2、pbm、pgm、ppm、pnm、pam、pcx
- **灵活的排序方式**: 支持按**文件名**、**修改时间**、**文件大小**进行正序或倒序排列，文件管理井井有条。

### 🌍 多语言支持
简体中文，繁体中文，英文，日文，韩语，法语

## ⌨️ 快捷键指南

为了提升浏览效率，青柠图片提供了丰富的键盘快捷键支持：

| 功能分类 | 按键 | 描述 |
| :--- | :---: | :--- |
| **基础导航** | `←` / `→` / `鼠标滚轮` | 切换上一张 / 下一张图片（长按可快速翻页） |
| | `Home` | 跳转至文件夹第一张图片 |
| | `End` | 跳转至文件夹最后一张图片 |
| | `Esc` | 关闭窗口 |
| **视图控制** | `+` / `-` | 快速放大 / 缩小软件窗口 |
| | `F11` | 开启 / 退出全屏模式 |
| | `L` | 打开当前图片所在文件夹并定位文件 |
| | `I` | 显示/隐藏图片 元数据面板 |
| **缩放模式** | `1` - `5` | 快速切换 5 种缩放模式（自动/焦点/中心/宽/高） |
| **排序控制** | `N` / `Shift + N` | 按**名称** 正序 / 逆序 |
| | `T` / `Shift + T` | 按**时间** 正序 / 逆序 |
| | `S` / `Shift + S` | 按**大小** 正序 / 逆序 |
