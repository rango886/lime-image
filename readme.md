# 青柠图片

> 🎬 **青柠图片** 是一款轻量级且高颜值的图片查看器。它集多格式支持与元数据编辑于一身，有五种图片缩放模式，配合人性化的快捷键操作，为您带来高效、流畅的沉浸式看图体验

## 功能特性
- **无边框设计**: 自动隐藏菜单，悬浮显示菜单栏
- **流畅的图片切换**:
- **五种图片缩放模式**:
    1. 自动缩放, 当图片宽>高，按宽填满软件窗口，当图片高>宽，按高填满软件窗口
	2. 焦点缩放锁定，适用于前后两张图片的分辨率一致，例如当图片在左上角缩放后，记住位置和缩放率，切到下一张，还是一样的缩放率和位置，如果是缩放在右下角，切换下一张，还是一样的缩放率和位置
	3. 中心缩放锁定，切换图片后记住缩放率，从图片中心缩放
	4. 宽度优先，缩放图片，宽度填满窗口
    5. 高度优先，缩放图片，高度填满窗口
- **支持多格式图片元数据编辑**:
- **窗口大小快速调整**:
- **支持多格式图片查看**:
- **多种排序方式**:
- **方便的快捷键**:
    +- 快速放大缩小窗口
	左右按键，按住不放一直切换
	home end 跳转到文件夹图片列表的开始和结束
	l 打开当前图片所在文件夹 标记文件
	n 按名称正序
	shift + n 按名称逆序
	t 按时间正序
	shift t 按时间逆序
	s 按大小正序
	shift s 按大小逆序
	i 显示文件的exif 信息
	f11 全屏
	esc关闭窗口
- **多语言支持**:
    简体中文，繁体中文，英文，日文，韩语，法语
