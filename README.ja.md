  <h1>Lime Image Viewer (青柠图片)</h1>

> **Lime Image Viewer** は、軽量で美しいデザインの画像ビューアです。多形式のサポートと強力なメタデータ編集機能を兼ね備え、5つの便利なズームモードと直感的なショートカットキー操作により、効率的でスムーズ、かつ没入感のある画像閲覧体験を提供します。

---

## ✨ 主な特徴

### 🎨 没入型ビジュアル体験
- **ボーダーレスデザイン**: 従来の煩雑なウィンドウ枠を排除し、メニューバーは自動的に隠れ、マウスをホバーさせた時だけ優雅に現れます。画面のすべてのピクセルを画像のために活用します。
- **スムーズな画像切り替え**: 画像読み込みロジックを最適化し、カクつきのない滑らかな操作感を実現しました。

### 🔍 便利なズームモード
さまざまな閲覧シーンに対応する5つのスマートズームロジックを提供します（**数字キー `1` - `5` で素早く切り替え可能**）：
1.  **スマート自動ズーム**: 画像のアスペクト比を自動識別します。横長の画像は幅に、縦長の画像は高さに合わせ、常にウィンドウ内に画像全体を表示します。
2.  **フォーカスロック（比較モード）**: 連写画像の選別や間違い探しに最適です。左上などの細部を拡大して確認している際、次の画像に切り替えても、同じズーム倍率と表示位置を維持するため、調整の手間が省けます。
3.  **センターロック**: 画像を切り替えても現在のズーム倍率を維持し、常に画面の中央を基準にレンダリングします。
4.  **幅優先**: ウィンドウの幅に合わせて強制的にズームします（縦読み漫画やウェブページのスクリーンショットに最適）。
5.  **高さ優先**: ウィンドウの高さに合わせて強制的にズームします。

### 🛠 強力な機能セット
- **多形式メタデータ編集**: 画像メタデータの閲覧だけでなく、編集もサポートしています。
- **ウィンドウサイズの高速調整**: ショートカットキーでウィンドウサイズを素早く調整でき、異なる解像度のデスクトップレイアウトに柔軟に対応します。
- **全フォーマット対応**: 対応形式: png, jpeg, jpg, bmp, gif, tif, tiff, webp, ico, jfif, heic, heif, avif, jxl, svg, psd, psb, dds, tga, xcf, hdr, exr, dng, cr2, cr3, nef, arw, orf, rw2, raf, sr2, srw, pef, x3f, erf, mef, mos, mrw, 3fr, kdc, jp2, j2k, jpf, jpx, jpm, mj2, pbm, pgm, ppm, pnm, pam, pcx
- **柔軟なソート機能**: **ファイル名**、**更新日時**、**ファイルサイズ**による昇順・降順の並べ替えに対応し、ファイル管理を整理します。

### 🌍 多言語サポート
簡体字中国語、繁体字中国語、英語、日本語、韓国語、フランス語

## ⌨️ ショートカットキーガイド

閲覧効率を高めるために、Lime Image Viewerは豊富なキーボードショートカットを提供しています：

| 機能分類 | キー | 説明 |
| :--- | :---: | :--- |
| **基本ナビゲーション** | `←` / `→` / `ホイール` | 前 / 次の画像へ切り替え（長押しで高速送り） |
| | `Home` | フォルダの最初の画像へ移動 |
| | `End` | フォルダの最後の画像へ移動 |
| | `Esc` | ウィンドウを閉じる |
| **表示コントロール** | `+` / `-` | ウィンドウサイズの拡大 / 縮小 |
| | `F11` | 全画面モードの切り替え |
| | `L` | 現在の画像の保存フォルダを開く |
| | `I` | メタデータパネルの表示 / 非表示 |
| **ズームモード** | `1` - `5` | 5つのズームモードを切り替え（自動/フォーカス/中央/幅/高さ） |
| **ソート制御** | `N` / `Shift + N` | **名前**順 昇順 / 降順 |
| | `T` / `Shift + T` | **日時**順 昇順 / 降順 |
| | `S` / `Shift + S` | **サイズ**順 昇順 / 降順 |
